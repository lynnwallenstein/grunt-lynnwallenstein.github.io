Auto Commenting of Partials

When working in a rails app, sometimes it is really easy to loose yourself in a complex path of nested partials. Not knowing where code is coming from or where the proper place is to edit the source can be extremly frustrating and after my consistent bitching, my friend Chris Maddox whipped up this really helpful gem that adds inline comments in your HTML showing you when partials start and end.

Install the gem as normal:

gem install partial_logging

and then view the source of your HTML in dev tools or however you view source (hint: the correct answer is devtools )

BOOM. Now you can easily track what is coming form where.
